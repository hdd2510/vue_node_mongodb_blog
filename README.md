# vue_node_mongodb_blog
项目说明：{
        1.前后端分离的个人博客.
        2.前端用vue.
        3.后端用node.
        4.数据库用mongod.
        
      }
项目运行：{
        1.git clone 项目
        2.cd vue_node_mongodb_blog
        3.npm install 
        4.cd server 
        5.npm install 
        6.npm run start
        7.cd ..
        8.npm run dev
      }  
	Npm install 
	进入server然后执行npm install
	前端文件 : npm run dev
后台文件 : npm run start
